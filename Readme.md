# Heyireeh - MERN eCommerce Machine Test

## Features
- **User Role:** Register/Login, View Products (Filter & Sort), Buy Now, Mock Payment, Order History.
- **Admin Role:** Category & Sub-category Management, Product Management, View All Orders.
- **Stock Control:** Automatic stock reduction upon successful mock payment.
- **Authentication:** Secure JWT-based auth with role-based access.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, Axios, React-Router-Dom
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** JSON Web Tokens (JWT)
